# heart-disease-Data--Analysis-project-EDA-
Heart Disease Risk - Data Analysis Project
What are we doing?
In this project we have a dataset with 5,500 patients. Each patient has health information like age, blood pressure, cholesterol etc. We want to understand this data and find out what causes high heart disease risk.

Step by Step Plan:
Load the data and look at it
Check if data has any problems
Look at each column one by one
Find patterns and relationships
Write our conclusions
